## ❓概要

その名の通り、これは**Rust**で開発された**美少女ゲーム**エンジンで、一般的なシーン切り替えとストーリー進行機能を備えています。

## ✅実装済み機能

* 背景画像
* テキスト
* ボイス
* BGM
* 立ち絵、表情
* 簡易アニメーション
* オートプレイ  
* スキップ  
* 選択肢
* バックログ・バックログジャンプ
* セーブ/ロード
* ウィンドウサイズ変更時のコンテンツ比率維持
* F11ショートカットでの全画面切り替え
